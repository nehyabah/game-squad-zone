# Scripts Archive

This directory contains one-time migration and data manipulation scripts that were used during development and initial data setup. These scripts are preserved for historical reference but are not part of the active codebase.

## Script Categories

### Data Addition Scripts
- `add-*-picks.js` - Scripts to add specific player picks for various weeks
- `add-*-week*.js` - Week-specific data addition scripts

### Verification Scripts
- `check-*.js` - Scripts to verify data integrity and game states
- `find-*.js` - Scripts to locate specific users or data

### Fix/Cleanup Scripts
- `fix-*.js` - One-time data correction scripts
- `clean-*.js` - Data cleanup and maintenance scripts
- `complete-*.js` - Scripts to finalize incomplete data

### Game Management
- `create-week*-games.js` - Scripts to generate weekly game schedules
- `update-*.js` - Score and result update scripts

## Note
These scripts were moved to archive on 2024-09-29 as part of project cleanup. They are kept for reference but should not be run against the production database without careful review.
